# engineDesign
工程设计是一门艺术，而不是一门科学  

## 学习思路
#### 如何快速的学习一个框架  
1. What: 框架是做什么用的，包含了哪些主要的功能和特性？
2. Why: 为什么要用这个框架、可以解决什么问题？
3. Where: 框架的应用场景（能做什么）？
4. How: 框架如何去使用?  
5. Design: 设计，框架的设计思路是什么？

#### 如何学习和讲解技术点
 3W+1H，  
 该技术要点是什么 (What)，  
 为什么(Why):为什么要用该技术，  
 应用场景(Where):该技术用在什么地方，  
 怎么做(How):该技术操作的时候怎么做
